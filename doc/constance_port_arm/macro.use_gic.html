[−][src]Macro constance_port_arm::use_gic

macro_rules! use_gic {
    (unsafe impl PortInterrupts for $sys:ty) => { ... };
}

Implement PortInterrupts, InterruptController, and Gic on the given system type using the General Interrupt Controller (GIC) on the target. Requires GicOptions.

Safety

  • The target must really include a GIC.
  • GicOptions should be configured correctly and the memory-mapped registers should be accessible.
